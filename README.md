## 🏪 محلك

[![version](https://img.shields.io/badge/version-1.2.0-blue.svg)](https://github.com/iofahmawi/mahalak)
[![PWA](https://img.shields.io/badge/PWA-Enabled-brightgreen)](https://iofahmawi.github.io/mahalak/)

نظام إدارة متكامل للمحلات التجارية الصغيرة. جميع البيانات تُحفظ على جهازك مباشرة دون إرسال أي معلومات لخوادم خارجية.

### 🔗 تجربة التطبيق
* [🚀 معاينة حية للتطبيق](https://iofahmawi.github.io/mahalak/)

### ✨ المميزات

* **المبيعات**: إنشاء فواتير مبيعات بسرعة مع دعم إلغاء الفاتورة وإعادة المخزون تلقائياً عند الإلغاء. يعرض البانر إحصائيات اليوم أو الإجمالي الكلي بضغطة واحدة.
* **المشتريات**: تسجيل فواتير الشراء مع تتبع أرقام الفواتير وربطها بالموردين.
* **المخزون بالدُفعات**: كل منتج يدعم دفعات متعددة بأسعار شراء وبيع مستقلة لكل دفعة مع عرض الربح المتوقع.
* **طرق الدفع**: دعم ثلاثة أنواع — نقدي، Visa/Mastercard، وآجل — في المبيعات والمشتريات مع إمكانية الفرز حسب نوع الدفع.
* **إدارة الموردين والزبائن**: متابعة ذمم الموردين والزبائن مع سجل تسديدات تفصيلي، إمكانية السداد الجزئي والكامل، وطباعة سند قبض فوري عند كل تسديد.
* **سجل الموردين وسجل الزبائن**: عرض كامل لجميع التسديدات مع فلتر حسب اليوم أو الفترة أو الكل، بحث بالاسم، وإمكانية طباعة أي سند في أي وقت.
* **سند القبض**: يُطبع تلقائياً عند كل تسديد ذمة ويتضمن اسم المحل، رقم السند، البيان مع رقم الفاتورة، المبلغ، وخط التوقيع.
* **المصاريف**: تسجيل المصاريف مع حقل المستفيد وطباعة سند صرف لكل مصروف يتضمن البيان وتوقيع المستلم.
* **سجل الصندوق**: جرد يومي للصندوق مع حساب الرصيد المتوقع تلقائياً — مع استثناء مبيعات ومشتريات البطاقة تلقائياً.
* **المسحوبات الشخصية**: تتبع السحوبات اليومية من الصندوق.
* **الإحصائيات**: تقارير بمربعات بصرية تشمل ملخصاً مالياً (إيرادات، تكلفة، ربح إجمالي، مصاريف، مسحوبات، صافي ربح)، تقارير المخزون، الأصناف، الزبائن، والصندوق.
* **الطباعة**: فواتير مبيعات ومشتريات ووصولات قبض وصرف جاهزة للطباعة مباشرة أو حفظها كـ PDF.
* **نظام المستخدمين**: دعم تعدد المستخدمين مع صلاحيات قابلة للتخصيص لكل مستخدم.
* **بحث سريع**: البحث في المبيعات والمشتريات والتسديدات بالاسم والتاريخ ورقم الفاتورة.
* **تصدير واستيراد شامل**: حفظ جميع البيانات بصيغة JSON مع إمكانية استعادتها — متاح للمدير فقط.
* **تنقل بالسحب**: التنقل بين تبويبات المبيعات والمشتريات والمخزون بالسحب الجانبي.
* **خصوصية تامة**: جميع البيانات تُخزَّن في المتصفح فقط دون رفع أي بيانات لخوادم خارجية.
* **تطبيق ويب تقدمي (PWA)**: يمكن تثبيت التطبيق على الجهاز والوصول إليه في أي وقت.

### 🛠 التقنيات المستخدمة

* **HTML5 & CSS3**: تصميم نقي وواضح دون أي أطر عمل.
* **JavaScript (Vanilla ES6+)**: لضمان أداء سريع وعمليات فورية.
* **IndexedDB**: لحفظ البيانات الضخمة محلياً على الجهاز.
* **localStorage**: لحفظ إعدادات التطبيق والمستخدمين.

### 🚀 كيفية الاستخدام

1. **فتح التطبيق**: اضغط على رابط التجربة أعلاه.
2. **إعداد المخزون**: اضغط على تبويب "المخزون" وأضف أصنافك مع أسعار الشراء والبيع.
3. **إنشاء فاتورة بيع**: اضغط "ابدأ البيع" واختر الأصناف وأدخل كمياتها ثم احفظ الفاتورة.
4. **تسجيل مشتريات**: انتقل لتبويب "شراء" وأضف فاتورة الشراء مع تحديد المورد وأصناف الدفعة.
5. **متابعة الذمم**: اضغط ☰ ثم "ذمم الموردين" أو "ذمم الزبائن" لمتابعة الذمم وتسجيل التسديدات وطباعة السندات.
6. **سجل التسديدات**: اضغط ☰ ثم "سجل الموردين" أو "سجل الزبائن" لعرض كامل تسديدات مع إمكانية إعادة طباعة أي سند.
7. **المصاريف**: اضغط ☰ ثم "تسجيل مصروف" لإضافة مصروف مع المستفيد وطباعة سند الصرف.
8. **سجل الصندوق**: اضغط ☰ ثم "سجل الصندوق" وأدخل الرصيد الأولي والجرد الفعلي لكل يوم.
9. **الإحصائيات**: اضغط ☰ ثم "إحصائيات" لعرض ملخص شامل بمربعات بصرية.
10. **نسخة احتياطية**: اضغط ☰ ثم "تصدير" لحفظ نسخة احتياطية كاملة من جميع البيانات.

### 📂 ملفات المشروع

* `index.html`: الملف الأساسي الذي يحتوي على الواجهة والمنطق البرمجي.
* `manifest.json`: ملف تعريف التطبيق للتثبيت على الأجهزة.
* `sw.js`: ملف Service Worker للتشغيل أوفلاين.
* `icon-1440.png`: أيقونة التطبيق.
